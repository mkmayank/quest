Continous Integration (CI)
    -   practice of merging all developer working copies to a shared mainline several times a day

WAR file (Web Application Resource or Web application ARchive)
    -   file used to distribute a collection of
        JAR-files, JavaServer Pages, Java Servlets, Java classes, XML files, tag libraries,
        static web pages (HTML and related files) and other resources that together constitute a web application.
